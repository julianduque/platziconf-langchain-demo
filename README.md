# PlatziConf LangChain Demo

Using LangChain.js to retrieve YouTube documents, store them in Weaviate as vectors and then have a contextual QA chain with them.

## Requisites

* Node.js LTS
* OpenAI API Key
* Weaviate Account or Local instance

## Install

```
npm install
```

### Load Documents

```
node load-docs.js
```

### Run demo

```
node demo-stuff.js
```
